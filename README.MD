1. Посмотреть темы и их ID:

````
python main.py --list-topics
````

2. Взять опрос в конкретной теме:

```
python main.py --topic-id 123
```

3. Найти тему по названию (например, “концерт”) и выбрать из списка:

```
python main.py --topic "концерт"
```

4. Найти нужный опрос по подстроке (твой пример):

```
python main.py --topic "концерт" --poll "Бал в Атриуме"
```
